# Безопасность

## Секреты

- Не коммитьте файлы **`.env`** с реальными паролями и `JWT_SECRET`.
- В репозитории должны быть только **`.env.example`** с плейсхолдерами.
- Для production смените пароли БД и сгенерируйте новый `JWT_SECRET` (≥ 32 символов).

## Сообщить о проблеме

Создайте приватный advisory в GitHub или issue с пометкой `security` (без воспроизведения секретов в открытом виде).
